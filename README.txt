Make sure all files are in the same directory and open 'index.html' in your browser.

This web application is a parody on WebMD
The data is read from an XML file and the attributes for each element must be in the same order as given. 
I'm using HTTP XML Request to load the data from the file, so a server or an IDE such as Webstorm may be required to run it.
I have included a second XML file to show the scalability
I keep track of displayed menus using a single array to allow the user to start over at any time
Once all menu options have been selected, you need to enter the details at the bottom to save the data
A search feature is provided to show that localStorage and cookies are implemented.
